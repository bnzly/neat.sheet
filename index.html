How to Use

      Configure League Settings:
      Use the League panel to select your league size, number of teams, weeks, and roster positions. Adjust each value using the +/– buttons or by typing directly into the field.
      Customize Scoring Rules:
      In the Scoring panel, set how points are awarded for any given stat. Modify any value to match your league's rules.
      Check Status & Get Your Draft Sheet:
      As you change settings, the Status area above the input form will automatically check if a draft sheet exists for your configuration. If available, a download link will appear. If not, you'll be prompted to submit your configuration.
